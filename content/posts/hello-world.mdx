## Welcome

Hello! I'm Richard. This blog is where I share my thoughts on software development, technology, and whatever else I find interesting.

<Callout type="tip">
  This blog is built with **Next.js 15**, **Velite**, and **Tailwind CSS v4**.
</Callout>

## What to expect

Here's what I plan to write about:

<Steps>
### Web Development
Practical guides on Next.js, React, TypeScript, and modern CSS.

### Open Source
Projects I'm working on and things I've learned along the way.

### Career & Learning
Reflections on growing as a software engineer.
</Steps>

## Tech Stack

This blog is powered by:

- **Next.js 15** — App Router with React Server Components
- **Velite** — Type-safe MDX content layer
- **Tailwind CSS v4** — CSS-first configuration
- **Zustand** — Lightweight UI state management
- **Prisma** — Type-safe database ORM for comments and reactions
